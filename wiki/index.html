Programma di compressione e decompressione file.

Programma principale. Programma per la compressione e la decompressione dei File mediante l'algoritmo di Huffman.

Nella fase di COMPRESSIONE il programma legge un file di testo dato in input e conta le occorrenze di ogni carattere e assegnando ai caratteri più frequenti codici più brevi. Al termine restituisce un file binario teoricamente compresso.

Nella fase di DECOMPRESSIONE il programma riceve in input il file binario compresso e mediante opportune operazione ricostruisce il codice di compressione e ricostruisce il file originario. Al termine restituisce il file originario in formato di testo.
